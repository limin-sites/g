> 不管是新系统和已安装好的系统，确保操作系统的更新是打开的，如果关闭了，很有可能 后面的配置都做不了。


#### 如果你也有遇到以下问题，试着按下面的方法解决

- Windows无法登陆微软帐号，如果遇到以下问题，确保系统更新是打开的。如果遇到下面的问题很可能要重装系统了
![输入图片说明](https://images.gitee.com/uploads/images/2021/0810/143319_fc9a30a6_325256.png "屏幕截图.png")

- 游戏服务无法更新，发现被禁用

无法更新的表现为：运行安装好的游戏时会自动跳到游戏服务商店页面，需要更新，但点更新却更新失败。可以打开系统服务以确认是否被禁用：
Win+R->services.msc->找到xbox game service->双击打开，看是否被禁用。如果能修改成自动运行，则修改好即可。如果是灰的无法修改，按下面的方法解决：

方法1: 从注册表中删除以下两项，打开注册表的方式：Win键+R->输入 `regedit`
```
]HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\GamingServices]
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\GamingServicesNet]
```

方法2:使用PowerShell命令重装游戏服务，操作方式： 按Win+S->在搜索框输入 `PowerShell`->右键：以管理员身份运行->依次输入下面的两个命令
```
get-appxpackage Microsoft.GamingServices | remove-AppxPackage –allusers
start ms-windows-store://pdp/?productid=9MWPM2CQNLHN
```

以上方法来自：https://docs.microsoft.com/en-us/answers/questions/409574/gaming-services-wont-installupdate-error-code-0x80.html

### 如何注册微软帐号， 安装、配置相关应用

- 如果还没有帐号请先按顼步骤注册微软帐号：
1. 打开网址: https://www.xbox.com/zh-cn/live  点击注册即可，如果已经有，直接登陆。
2. 地区选国外都可以。通常选HK、TW、US,或者低价区。
3. microsoft account profile 和xbox profile 
   实际上是同一个帐号，前一个用于微软的其他服务(包含xbox订阅管理)，xbox帐号用于游戏，个人信息是分开的，也就是xbox live的个人信息单独存在、更新。  xbox profile地址：https://account.xbox.com/zh-hk/Profile  microsoft account 地址：https://account.microsoft.com
   ![输入图片说明](https://images.gitee.com/uploads/images/2021/0727/110517_9cfc8baa_325256.png "屏幕截图.png")
   登陆xbox站点后，右上角可切换，如上图所示。

- 打开windows商店，确保已安装以下应用，且已更新到最新版。除了Xbox Accessories，其他应用Windows系统自带，不需要单独安装。
1. Xbox APP(其实是用来查看和下载xgp的游戏)，商店地址如下：
https://www.microsoft.com/zh-hk/p/xbox/9mv0b5hzvk9z
1. Xbox 控制台小帮手，原Xbox APP，这里面可查看成就、好友、Xbox捕获，搜索游戏, 安装游戏到Xbox，串流Xbox主机游戏，类似于xbox主机上的主界面。
1. Xbox Game Bar, 按西瓜键会显示这个工具，在玩游戏时查看好友，邀请好友，调整音量，创建派对，录屏和捕获等等，相当于游戏辅助类
https://www.microsoft.com/zh-hk/p/xbox-game-bar/9nzkpstsnw4p?activetab=pivot:overviewtab
1. Xbox Accessories：Xbox手柄/外设小助手，用于手柄或其他官方外设的一些配置,商店地址如下：
https://www.microsoft.com/zh-cn/p/xbox-accessories/9nblggh30xj3?activetab=pivot:overviewtab
