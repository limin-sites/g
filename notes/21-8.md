### 怎么样和Xbox主机联机，如何识别Xbox游戏是否支持跨平台联机？

这里也要注意一下，最开始可能会总联不上机，我们得检查一下Xbox Live的网络状态，一般来说NAT为严格时，多人联机不能做主机，最好是开放，或中等。

如果联机有问题，打开 设置->游戏->Xbox网络, 等检测完后，如果Teredo字样，说明有问题，这个时候要确保以下几点：

- 路由器，最好直连接着公网的路由器，不要搞双重路由，搞双重路由NAT基本不可能为开放，会提示双重NAT
- 路由器的NAT和端口映射要打开，这样联机时数据才能正常发送。
- 有条件的把DMZ打开，放Xbox暴露在公网上
- 如果网络支持IPV6，那尽量打开吧
- 准备加速器，大把分游戏需要加速器，加速器不急着用，可以在试玩后在决定是否要使用，如果裸连能正常玩，那开不开无所谓

最后需要确保：Xbox Game Bar 要处理打开状态，否则 可能 会收到不到朋友 发的游戏邀请。

**如何识别Xbox游戏是否支持跨平台联机？**

跨平台联机分好几种情况：

- xbox主机与PC 商店版游戏联机：支持xpa的估计都支持，可以在游戏的多人模式处可获悉，见下图，如果是同一个帐号下的订阅，同一个帐号只能登录一个设备，可以一端用小号登陆即可同时玩。
![xbox跨平台联机](https://images.gitee.com/uploads/images/2021/1115/105842_05593231_325256.png "屏幕截图.png")
- xbox主机+PC 商店版游戏+Steam联机：通常要看游戏是否支持
- xbox主机+PC 商店版游戏+Steam PC版或Epic PC版联机：通常要看游戏是否支持
- xbox主机+PC 商店版游戏+Steam PC版联机或Epic PC版+PS版联机：通常要看游戏是否支持

- 游戏服务无法更新，发现被禁用；或者游戏无法更新

无法更新的表现为：运行安装好的游戏时会自动跳到游戏服务商店页面，需要更新，但点更新却更新失败。可以打开系统服务以确认是否被禁用：
Win+R->services.msc->找到xbox game service->双击打开，看是否被禁用。如果能修改成自动运行，则修改好即可。如果是灰的无法修改，按下面的方法解决：

方法1: 从注册表中删除以下两项，打开注册表的方式：Win键+R->输入 `regedit`
```
]HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\GamingServices]
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\GamingServicesNet]
```

方法2:使用PowerShell命令重装游戏服务，操作方式： 按Win+S->在搜索框输入 `PowerShell`->右键：以管理员身份运行->依次输入下面的两个命令
```
get-appxpackage Microsoft.GamingServices | remove-AppxPackage –allusers
start ms-windows-store://pdp/?productid=9MWPM2CQNLHN
```

以上方法来自：https://docs.microsoft.com/en-us/answers/questions/409574/gaming-services-wont-installupdate-error-code-0x80.html
